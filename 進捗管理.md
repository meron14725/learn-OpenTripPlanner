- [x] **#1-2** 直通運転（interlineWithPreviousLeg）確認 — [詳細](開発計画.md#1-2-直通運転interlinewithpreviousleg確認)

- [x] **#1-3** stopCalls / 途中停車駅フィールド確認 — [詳細](開発計画.md#1-3-stopcalls--途中停車駅フィールド確認)

- [x] **#1-5** Nominatim（住所→座標）動作確認 — [詳細](開発計画.md#1-5-nominatim住所座標動作確認)
