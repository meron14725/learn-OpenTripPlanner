## POST `/api/v1/routes`

**目的**: 出発駅と到着駅、到着希望時刻を指定して、最適な電車経路を検索する。

### リクエスト

```json
{
  "origin": "高円寺",
  "destination": "渋谷",
  "desired_arrival_time": "2026-03-10T09:00:00+09:00"
}
```

| フィールド | 型 | 必須 | 説明 |
|-----------|----|----|------|
| `origin` | string | ✅ | 出発駅名（例: `"高円寺"`） |
| `destination` | string | ✅ | 到着駅名（例: `"渋谷"`） |
| `desired_arrival_time` | datetime | ✅ | 到着希望時刻（ISO 8601 形式） |

### レスポンス（200 OK）

```json
{
  "board_station": "高円寺",
  "depart_at": "08:43",
  "arrive_at": "08:57",
  "num_transfers": 1,
  "legs": [
    {
      "mode": "WALK",
      "line_name": null,
      "from_station": "Origin",
      "to_station": "高円寺",
      "departure_time": "08:38",
      "arrival_time": "08:43"
    },
    {
      "mode": "RAIL",
      "line_name": "中央線快速",
      "from_station": "高円寺",
      "to_station": "新宿",
      "departure_time": "08:43",
      "arrival_time": "08:50"
    },
    {
      "mode": "RAIL",
      "line_name": "埼京線・川越線",
      "from_station": "新宿",
      "to_station": "渋谷",
      "departure_time": "08:52",
      "arrival_time": "08:57"
    },
    {
      "mode": "WALK",
      "line_name": null,
      "from_station": "渋谷",
      "to_station": "Destination",
      "departure_time": "08:57",
      "arrival_time": "08:57"
    }
  ],
  "message": "08:43 に 高円寺 を出発してください"
}
```

| フィールド | 型 | 説明 |
|-----------|----|----|
| `board_station` | string | 最初に乗車する駅名 |
| `depart_at` | string | 最寄り駅の出発時刻（HH:MM） |
| `arrive_at` | string | 目的地への到着時刻（HH:MM） |
| `num_transfers` | int | 乗り換え回数 |
| `legs` | array | 区間ごとの経路情報 |
| `message` | string | ユーザー向けの案内メッセージ |

#### legs の各要素

| フィールド | 型 | 説明 |
|-----------|----|----|
| `mode` | string | 移動手段（`WALK` / `RAIL` / `SUBWAY` / `BUS`） |
| `line_name` | string \| null | 路線名（徒歩区間は null） |
| `from_station` | string | 出発地点 |
| `to_station` | string | 到着地点 |
| `departure_time` | string | 出発時刻（HH:MM） |
| `arrival_time` | string | 到着時刻（HH:MM） |

## POST `/api/v1/routes/departure-time`

**目的**: 身支度時間（`preparation_minutes`）を考慮して「何時に家を出ればいいか」を返す。

### リクエスト

```json
{
  "origin": "高円寺",
  "destination": "渋谷",
  "desired_arrival_time": "2026-03-10T09:00:00+09:00",
  "preparation_minutes": 15
}
```

| フィールド | 型 | 必須 | 説明 |
|-----------|----|----|------|
| `origin` | string | ✅ | 出発駅名 |
| `destination` | string | ✅ | 到着駅名 |
| `desired_arrival_time` | datetime | ✅ | 到着希望時刻 |
| `preparation_minutes` | int | — | 身支度にかかる時間（分）。デフォルト: 0 |

### レスポンス（200 OK）

```json
{
  "leave_home_at": "08:28",
  "board_station": "高円寺",
  "board_at": "08:43",
  "message": "08:28 に家を出発してください（身支度15分を含む）"
}
```

| フィールド | 型 | 説明 |
|-----------|----|----|
| `leave_home_at` | string | 家を出るべき時刻（HH:MM） |
| `board_station` | string | 最寄り駅名 |
| `board_at` | string | 最寄り駅の電車出発時刻（HH:MM） |
| `message` | string | ユーザー向けの案内メッセージ |

## エラーレスポンス

全エンドポイント共通のエラー形式：

```json
{
  "error": {
    "code": "STATION_NOT_FOUND",
    "message": "未対応の駅名: '荻窪' — backend/services/geocoder.py の STATION_COORDS に追加してください"
  }
}
```

### エラーコード一覧

| code | HTTP | 状況 | 対処法 |
|------|------|------|--------|
| `STATION_NOT_FOUND` | 400 | 未登録の駅名が入力された | 対応駅一覧を確認するか、駅名辞書に追加 |
| `ROUTE_NOT_FOUND` | 404 | 経路が見つからない | 日時や区間を変更して再試行 |
| `OTP_UNAVAILABLE` | 502 | OTP2 サーバーに接続できない | OTP2 が起動しているか確認 |

## 対応駅

GTFS データから自動インポートした **1,541駅** に対応しています。

JR東日本、東京メトロ、都営地下鉄、東急、京王、小田急、西武、東武、京急、京成、相鉄、横浜市営地下鉄、みなとみらい線、ゆりかもめ、東京モノレール、多摩モノレール、つくばエクスプレス、北総鉄道、新京成、埼玉高速鉄道、東葉高速鉄道、りんかい線の各路線駅が利用可能です。

> 💡 駅名の末尾に「駅」を付けても自動で除去されます（例: `"渋谷駅"` → `"渋谷"` として処理）

> ⚠️ GTFS データに含まれていない駅名を指定すると `STATION_NOT_FOUND` エラーになります。

## curl での実行例

### 基本的な経路検索

```bash
curl -s -X POST http://localhost:8000/api/v1/routes \
  -H "Content-Type: application/json" \
  -d '{
    "origin": "高円寺",
    "destination": "渋谷",
    "desired_arrival_time": "2026-03-10T09:00:00+09:00"
  }' | python3 -m json.tool
```

### 身支度時間を考慮した出発時刻

```bash
curl -s -X POST http://localhost:8000/api/v1/routes/departure-time \
  -H "Content-Type: application/json" \
  -d '{
    "origin": "池袋",
    "destination": "東京",
    "desired_arrival_time": "2026-03-10T10:00:00+09:00",
    "preparation_minutes": 20
  }' | python3 -m json.tool
```

### 存在しない駅を指定（エラー例）

```bash
curl -s -X POST http://localhost:8000/api/v1/routes \
  -H "Content-Type: application/json" \
  -d '{
    "origin": "荻窪",
    "destination": "渋谷",
    "desired_arrival_time": "2026-03-10T09:00:00+09:00"
  }' | python3 -m json.tool
```

## システム構成

```
[クライアント（curl / フロント）]
        │ POST /api/v1/routes
        │ { origin, destination, desired_arrival_time }
        ▼
[FastAPI バックエンド :8000]
  ├─ Geocoder: 駅名 → 緯度経度（20駅の座標辞書）
  └─ OTP2 GraphQL クライアント
        │ planConnection(latestArrival=...) ← 到着時刻指定
        ▼
[OTP2 Java サーバー :8080]
  └─ data/ の GTFS + OSM PBF でグラフ構築済み
        │ 経路・時刻データ
        ▼
[FastAPI バックエンド]
        │ レスポンス整形
        ▼
{ "board_station": "高円寺", "depart_at": "08:43", ... }
```
